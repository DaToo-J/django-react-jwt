# Usage

The notes of backend and frontend are in the corresponding folders.

## Backend
1. `cd backend`;
2. `pip install -r requirements.txt`;
3. `python manage.py createsuperuser`;
4. `python manage.py runserver`;

## Frontend
1. `cd frontend`;
2. `npm install`;
3. `npm start`;
4. open browser http://localhost:3000;
